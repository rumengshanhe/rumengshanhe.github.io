网络操作

开始之前，先了解一下Windows下和Linux下各自的基本网络操作，以备后用；

Linux的网络命令比较多，Linux网络操作命令的一个特点是，命令参数选项和功能很多，一个命令往往还可以实现其它命令的功能，许多细节以自行查询为宜。

查看IP地址

Windows下

得到Windows下主机IP地址为192.168.0.106；

Linux下

Linux虚拟机的网络设置有两个问题需要注意

注意问题

问题1：

确保虚拟机的网络为桥接方式!!!

问题2：

确保网卡处于激活状态!!!

查看方法如下：

  • 输入命令：system-config-network
  • 会出现弹窗，检查eth0是否为活跃状态，如果不是活跃状态，激活即可。

查看ip地址

命令：ifconfig

得到Linux下的主机IP地址为192.168.10.183

测试连通性

重启网络

service network restart

测试连通性

在Windows命令行中，看是否可以ping通Linux主机。

ping 192.168.10.183

其他操作

临时指定IP

ifconfig eth0 192.168.0.?(10-253)

注意：同一局域网内IP地址不能重复

终止命令执行组合键

ctrl + c

从服务器下载资源

测试网络是否连通

ping 192.168.0.254 -c 3

192.168.0.254是服务器的IP地址，关于如何在个人主机上搭建服务器，在下一篇详述。

查看共享文件

[root@localhost linux]# showmount -e 192.168.0.254

Export list for 192.168.0.254:

/example 1 *

/example 2 *

/example 3 *

上述三个目录便是服务器端分享的文件目录

挂载

挂载共享目录到本机

mount 192.168.0.254:/27_course /mnt

下载到本机

解挂

umount /mnt

实现NFS服务

NFS ：Network File System，即网络文件系统。是由Sun公司开发的一个基于内核的文件系统，基于RPC（远程过程调用协议）实现。通过使用NFS服务，用户和程序可以像访问本地文件一样访问远端系统上的文件。
