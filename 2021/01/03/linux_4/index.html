第四篇--实现NFS服务

使用 NFS 服务实现 Linux 间文件共享

NFS ：Network File System，即网络文件系统。是由Sun公司开发的一个基于内核的文件系统，基于RPC（远程过程调用协议）实现。通过使用NFS服务，用户和程序可以像访问本地文件一样访问远端系统上的文件。

一、服务器端配置

1. 获取镜像文件

把系统镜像存放到 /tools/ 目录下，检查是否已存在

[root@localhost linux]# ls /tools/rhel55.iso

2. 挂载镜像文件

在根目录下创建 /iso 文件夹作为挂载点。

mkdir /iso

mount -t iso9660 -o loop /tools/rhel55.iso /iso

3. 配置yum

打开yum配置文件

/etc/yum.repos.d/rhel-debuginfo.repo

含义

文件内容 含义
[rhel-debuginfo] 表示标签
name=Red Hat Enterprise Linux $releasever - $basearch - Debug 表示名字
baseurl=ftp://ftp.redhat.com/pub/redhat/linux/enterprise/。。。 表示路径
file:// 表示本地文件
enabled=0 0 表示关闭、1 表示开启
gpgcheck=1 0 表示不校验、1 表示校验

4. 重载

清空以前配置并重新加载现在配置

yum clean all

5. 安装NFS服务

命令

安装命令 yum install nfs*

卸载命令 yum erase nfs*

6. 配置 NFC 服务

配置文件

在 /etc/ 目录下创建配置文件

vim /etc/exports

然后按照以下格式输入内容。

格式

共享目录名(必须是绝对路径) client(option)

client :

* : 表示所有可以ping通的客户端

option :

ro : 表示只读

rw : 表示可读可写

7. 重启 NFS 服务

命令

service nfs restart

二、客户端配置

1. 检测网络是否连通

service network restart

2. 查看共享文件

showmount -e service_ip

3. 挂载

mount service_ip:sharedir /mnt

4. 拷贝

cp /mnt /dest_dir

5. 解挂

umount /mnt

end
