第六篇 简单 Shell 编程

shell 是用于侦听用户输入指令的工具，其接受到这些指令，然后执行相应的操作；shell是一种计算语言，主要用于脚本维护。

shell 环境

shell的变量是没有类型的，但是有环境变量和局部变量之分

环境变量

即全局变量，可以被当前shell或者子shell访问；

查看环境变量

env

定义环境变量

export a=hello

访问变量值

${variable_name}

局部变量

局部变量只能被当前的shell所访问

查看局部变量

set

定义局部变量

b=123

访问变量

${variable_name}

算术运算符

+ - * / %

format

$[$variable_name + $variable_name]

shell 的语句

输出语句

format

echo [option] "hello"

option

  • -n : 表示不换行

输入语句

format

read [option] variable_name

option

  • -p : 表示显示提示信息
  • -s : 表示以非显式方式接受输入
  • -n : 表示指定接受字符个数
  • -t : 表示超时时间（秒）
