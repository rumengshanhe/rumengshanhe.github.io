简单网络操作和文件操作

一、网络操作

查看IP地址

1. Windows

命令：
ipconfig /all

得到Windows下主机IP地址为192.168.0.106；

2. Linux

  • Linux虚拟机有两个问题需要注意

问题1：

确保虚拟机的网络为桥接方式!!!

问题2：

确保网卡处于激活状态!!!

查看方法如下：

  • 输入命令：system-config-network
  • 会出现弹窗，检查eth0是否为活跃状态，如果不是活跃状态，激活即可。
  • 查看ip地址

    命令：ifconfig

    得到Linux下的主机IP地址为192.168.10.183

测试连通性

1. 重启网络

service network restart

2. 测试连通性

在Windows命令行中，看是否可以ping通Linux主机。

ping 192.168.10.183

其他操作

1. 临时指定IP

  • ifconfig eth0 192.168.0.?(10-253)

    注意：同一局域网内IP地址不能重复

2. 终止命令执行组合键

  • ctrl + c

如何从服务器下载资源

1. 测试网络是否连通

ping 192.168.0.254 -c 3

192.168.0.254是服务器的IP地址，关于如何在个人主机上搭建服务器，在下一篇详述。

2. 查看共享文件

[root@localhost linux]# showmount -e 192.168.0.254

Export list for 192.168.0.254:

/tools *

/27_course *

/27_homework *

上述三个目录便是服务器端分享的文件目录

3. 挂载

挂载共享目录到本机

mount 192.168.0.254:/27_course /mnt

4. 下载到本机

mkdir /zhangsan //创建一个下载目录

\cp /mnt/* /zhangsan -rf //拷贝

5. 解挂

umount /mnt

文件操作

ls

format:
    ls [OPTION]... [FILE]...

option :
    -a : 表示显示所有信息(包括 . .. 和隐藏文件或者目录)
        . : 表示当前工作目录
        .. : 表示上级目录
        .filename : 表示隐藏文件
    -i : 显示文件索引号
        计算机识别文件的唯一标识是inode节点号 文件索引号
    -l : 表示长格式显示文件信息
        [root@localhost test]# ls -l
        总计 16
        -rwxr-xr-x 1 root root 4729 11-29 20:18 a.out
        -rw-r--r-- 1 root root    6 11-29 20:13 a.txt
        -rw-r--r-- 1 root root   78 11-29 20:18 hello.c
        第一列：表示文件类型和权限
            第一个字符：表示文件类型
                - : 表示普通文件
                d : 表示目录
                c : 表示字符设备类文家
                b : 表示块设备文件
                l : 表示链接文件
                p : 表示管道文件
                s : 表示套接子文件
            第二到第十位 表示权限
                r : 表示只读    4
                w : 表示只写    2
                x : 表示可执行    1
                - : 表示无权限    0
            三位为一组。共三组
            第一组：表示属主权限 u
            第二组：表示属组权限 g
            第三组：表示其他用户权限 o

            指定权限命令：chmod
                原有基础上添加权限 +
                    案例：
                        [root@localhost test]# ll
                        总计 0
                        -rw-r--r-- 1 root root 0 12-04 20:34 aa
                        [root@localhost test]# chmod u+x aa
                        [root@localhost test]# ll
                        总计 0
                        -rwxr--r-- 1 root root 0 12-04 20:34 aa
                原有基础上取消权限 -
                    案例：
                        [root@localhost test]# ll
                        总计 0
                        -rwxr--r-- 1 root root 0 12-04 20:34 aa
                        [root@localhost test]# chmod u-w aa
                        [root@localhost test]# ll
                        总计 0
                        -r-xr--r-- 1 root root 0 12-04 20:34 aa

                直接指定文件的权限 =
                    案例：
                        [root@localhost test]# chmod  u=rwx aa
                        [root@localhost test]# ll
                        总计 0
                        -rwxr--r-- 1 root root 0 12-04 20:34 aa
                一次性指定权限：
                    案例：
                        [root@localhost test]# ll
                        总计 0
                        -rwxr--r-- 1 root root 0 12-04 20:34 aa
                        [root@localhost test]# chmod u=rx,g+w,o-r aa
                        [root@localhost test]# ll
                        总计 0
                        -r-xrw---- 1 root root 0 12-04 20:34 aa
            修改文件属主或者属组
                chown username:groupname filename
                    案例：
                        [root@localhost test]# chown tom aa
                        [root@localhost test]# ll
                        总计 4
                        ---------- 1 tom root 19 12-04 20:48 aa
                        [root@localhost test]# chown :jim aa
                        [root@localhost test]# ll
                        总计 4
                        ---------- 1 tom jim 19 12-04 20:48 aa
                        [root@localhost test]# chown jim:tom aa
                        [root@localhost test]# ll
                        总计 4
                        ---------- 1 jim tom 19 12-04 20:48 aa
            创建用户：
                useradd zhangsan 表示创建一个zhangsan的用户
                passwd zhangsan  设定用户密码



        第二列：表示文件硬连接数
            硬链接：
                创建：
                ln destfile srcfile
                    1、 给文件做一个备份
                    2、 硬链接文件inode节点号一致
                    3、 不可以对目录备份
                    4、 不可以跨分区
            软连接：
                创建：ln -s destfile srcfile
                    1 相当于创建一个快捷方式
                    2 文件inode节点号不一致
                    3 可以对目录设置
                    4 不可以跨分区


        第三列：表示文件属主 文件拥有者

        第四列：表示文件属组 文件拥有组

        第五列：表示文件大小 字节
            -h : 表示易读方式显示文件大小
                bit => byte => K => M => G => T => P => ....
        第六列：表示修改文件内容时间
            atime:访问时间 (cat tail head more less ...)
            ctime:表示修改文件属性时间(chmod chown)
            mtime:表示内容时间 同时触发atime 和 ctime时间

        第七列：表示文件名

cd

切换工作目录

绝对路径： 从根开始路径就是绝对路径
相对路径： 从当前工作目录开始就是相对路径

.     : 表示当前工作目录
..    : 表示上级目录
~    : 表示当前用户家目录
    区分用户是通过用户uid来区分
    [root@localhost bb]# id root
    uid=0(root) gid=0(root)
    groups=0(root),1(bin),2(daemon),3(sys),4(adm),6(disk),10(wheel)
    [root@localhost bb]# id tom
    uid=500(tom) gid=500(tom) groups=500(tom)

    管理员用户：root
        /root
    普通用户：
        /home/
cd : 表示回到当前用户家目录下

- : 表示返回上次所处的目录环境

[root@localhost test]# pwd查看当前工作目录绝对路径
/test

touch

创建文件

touch filename

touch a{1..100} //创建了100个文件

rm

删除

rm option filename

option:
    -f : 表示无序确认删除
    -r : 表示删除目录

通配符：
    * : 匹配任意多个字符
    ? : 匹配任意一个字符

mkdir

表示创建目录

[root@localhost test]# mkdir aa/bb/cc/dd

mkdir: 无法创建目录 “aa/bb/cc/dd”: 没有那个文件或目录

[root@localhost test]# mkdir -p aa/bb/cc/dd

tree

树状结构显示目录

rmdir

删除目录，不能删除非空目录
