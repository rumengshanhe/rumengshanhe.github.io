第二篇--Linux基础知识

Linux基础知识

一、linux介绍

定义

Linux–是一种类unix操作系统，确切的说是指操作系统的内核(kernel)。Linux是自由软件和开放源代码软件发展中最著名的例子。

Linux 发行版本

  • redhat : 公司运营，提供完整的解决方案，更专业
  • fedora : 和Ubuntu的包管理不一样，Fedora对新软件及新技术吸纳比较快
  • debian : 最遵循GNU规范操作系统，Ubuntu就是基于Debian的
  • Ubuntu : 国内乃至全球热门的Linux发行版，安装简单，社区活跃
  • Centos : 特点是相当稳定，版本更新紧跟RedHat，非常适合作为服务器操作系统使用
  • Suse linux : 德国著名linux，被称为“最华丽的Linux发行版”
  • 红旗Linux : 由中国中科红旗软件技术有限公司开发,方便国内的用户使用
